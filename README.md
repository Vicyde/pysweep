# Pysweep

A minesweeper clone in Python.

CLI-version

## How To Play

When starting the game, it'll ask for a width, height and percentage of mines.

For the percentage it is suggested to keep it around 10%. Use higher numbers at
your own risk!


When the game asks for coordinates, you can either enter the X Y pair, i.e.:

4 3

For position 4,3. Or preceed this with M in order to mark the coordinate:

M 4 3

The game is over when either all cells without a mine are opened, or all mines
are marked.
